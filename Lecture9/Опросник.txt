Кто-то зашёл на сайт
Выводим приветствие
Выводим Суть(вопрос, тематика) опросника
Предлогаем принять участие в опросе (3 кнопки (Авторизация, Просмотр текущего результата, Без авторризации))

Если выбор (авторизация)
	Редирект на форму аворизации (2 поля ( э-майл, пароль) 2 кнопки(Авторизация, отмена))
		Если (Авторизация)
			проверка в базе на сущствующего юзера
				тру - 
					через метод пост отправка данных
				Проверка на наличие куки, и ИП в базе(если уже проходил)
					нету -
						 редирект на опросник, сохранение в сессии
					тру - 
						вывод из базы сохранёного результата
				фалс - 



					 -вывод ошибки  и  предложение рагистрации
		Если (Отмена)
			Возврат на главную страницу с топиком

Если вывод результата
	Создаём диаграмму (таблицу) из результатов подсчёта данных и количество прочитанных данных
	Кнопка возврата на главную

Если без авторизации
	Проверка на наличие куки, и ИП в базе(если уже проходил)
		если проходил
			вывод его результата
		нет
			редирект на опросник и присвоение ему имени(userid=."целочисленное значени текущего комп времени/80 + ранд (0,10)")

Опросник
		открытие сессии, в сессию помещается (эмайл либо ИД пользователя)
	чтение файла с вопросами
	чтение файлов с ответами 
	циклический вывод вопросов и ответов к ним
		ответы выводятся с radiobutton
	перед сохранением проверяется наличие ответов  в каждой секции
		нету - 
			вывод ошибки о незаполненности формы
		есть -
			проверка отправляемых результатов (наличие ключа ответа в файле ответа для вопроса) 
				тру -
					отправка результатов с каждой секции с переменну
					создание файла с именем (1я часть имейла_2я часть э-мейла)
					запись результатов
					отображение результатов и кнопки сохранить
				при нажатии на сохранить
					сохранение данных о пользователе (мыло или идентифиатор в куки с названием файла ответов)
					редирект на общие результаты
				фалсе - 
					вывод сообщения об ошибке
					обнулить все значения (либо редирект на начало (на пустой опросник, обнуление сессии))
